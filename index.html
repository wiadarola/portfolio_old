Projects

Monocular Depth Detection Using a CNN

This project use a Convolutional Neural Network to estimate depth images from standard 2D RGB images. By training the network with paired 2D and depth camera images, the model can now infer depth details for any given 2D image. This depth perception capability has potential applications in robotics, AR, and various computer vision tasks.

Dataset

Magic the Gathering Card Price Predictor

This project uses regression techniques to estimate the price of Magic the Gathering creature-typed cards. It further classifies each card by its value, determining if it's priced above or below $1. This categorization aids players in budget-focused decisions before the release of a new expansion.

Repository
Dataset

15-Puzzle Solver with A* Search

An AI project to tackle the 15-puzzle, a classic sliding tile puzzle. Leveraging the A* search algorithm, the solution efficiently reads puzzle configurations from a text file and computes optimal solutions, making it a showcase of effective pathfinding and graph traversal techniques in AI.

Repository
15-Puzzle

Futoshiki Puzzle Solver with Backtracking & CSPs

An AI solution to navigate the Futoshiki puzzle, leveraging the principles of Constraint Satisfaction Problems (CSPs) and backtracking search. The solver efficiently handles puzzle configurations and determines optimal solutions, utilizing depth-first search and constraint satisfaction techniques in AI.
